Módulos do Curso Vue.js Expert

Mais de 9 módulos práticos que cobrem tudo que você precisa saber sobre Vue.js

Módulo 1 - Introdução ao Vue.js

Configuração de Ambiente: Setup do Node.js, npm/yarn e extensões. Primeiro Projeto Vue.js: Vue CLI/Vite e conceitos fundamentais. Projeto via CDN: Vue.js direto via CDN para prototipagem.

Palavras-chave: Vue 3, Setup, CDN

Módulo 2 - Fundamentos do Vue.js

Template Syntax: Interpolação. Diretivas: v-if, v-for. Reatividade: refs, reactive. Computed: Propriedades. Lifecycle: Ganchos. Watchers: Observadores. Template Refs: Referências. Composition API: API moderna.

Palavras-chave: Reatividade, Diretivas, Lifecycle

Módulo 3 - Componentes Vue.js

Introdução aos Componentes: Conceitos fundamentais. Single File Component: Estrutura .vue (SFC). Slots & Props & Events: Comunicação entre componentes. v-model & defineModel: Two-way binding. Importação Assíncrona: Lazy loading de componentes.

Palavras-chave: SFC, Props, Events, v-model

Módulo 4 - MVP com DaisyUI em Vue.js

Introdução ao DaisyUI: Framework CSS baseado em Tailwind. Componentes DaisyUI: Cards, forms, modals e navigation. Criando MVP Completo: Aplicação funcional do zero. Temas e Customização: Dark mode e temas personalizados. Responsividade: Design adaptativo e mobile-first.

Palavras-chave: DaisyUI, MVP, Tailwind, Responsive

Módulo 5 - Vuetify com Vue.js

Introdução ao Vuetify: Conceitos e Material Design. Componentes Vuetify: Cards, forms, navigation, etc. Temas e Customização: Cores e estilos personalizados. MVP com Vuetify: Aplicação completa Material Design. Build & Deploy: Produção e hospedagem.

Palavras-chave: Vuetify, Material Design, MVP, Deploy

Módulo 6 - Quasar Framework e Vue.js

Preparação do Ambiente: Setup e configuração inicial. Quasar CLI: Comandos e funcionalidades. Criando Projeto: Estrutura e organização. Componentes Quasar: Biblioteca completa de componentes. Plugins Quasar: Extensões e funcionalidades. MVP Completo: Aplicação real com Quasar. Deploy & Build: Netlify e outros serviços.

Palavras-chave: Quasar, CLI, Components, Deploy

Módulo 7 - Nuxt e Vue.js

Introdução ao Nuxt: Framework full-stack para Vue.js. Criando Projeto com Nuxt: Setup e estrutura inicial. Componentes com Nuxt: Auto-imports e organização. Páginas e Roteamento: File-based routing automático. Layouts: Templates compartilhados. Middleware: Autenticação e proteção de rotas. SSR & Deploy: Server-side rendering e hospedagem.

Palavras-chave: Nuxt, SSR, Routing, Middleware

Módulo 8 - Data Visualization com Vue.js

Gráficos com Vue.js: Bibliotecas de gráficos com Vue.js. Criando Projeto com Vue.js: Setup e estrutura inicial. Echarts com Vue.js: configurações e inicialização. Criando Componente de Gráfico: componente personalizado de gráfico. Apexcharts: Configurando biblioteca apexcharts. Dashboard: Criando um dashboard com os gráficos.

Palavras-chave: Apexcharts, Echarts, Gráficos, Vuejs

Módulo 9 - PrimeVue com Vue.js

Introdução e Instalação: Instalando a biblioteca em um projeto vite. Configurando tema: Configure seu tema personalizado. Introdução a componentes: Desvente os compomentes do Primevue. Criando MVP: Crie um MVP real para seu portfólio. Deploy: Configure o Deploy da aplicação. Otimização: Otimize sua aplicação para melhor performance.

Palavras-chave: Primevue, Temas, Componentes, MVP
